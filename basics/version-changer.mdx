## Minecraft-Version ändern

Wenn du eine andere Minecraft-Version für deinen Server nutzen möchtest, kannst du das ganz einfach über das Panel erledigen.

### Panel öffnen

- Gehe auf https://panel.craftinghosting.com.
- Melde dich mit deinen Zugangsdaten an.

### Server auswählen

- Nach der Anmeldung siehst du deine vorhandenen Server.
- Klicke auf den Server, dessen Version du ändern möchtest.

### Version Changer finden

- In der linken Navigationsleiste gibt es den Bereich **MANAGEMENT**.
- Klicke dort auf **Version Changer**.

### Version auswählen

- Im **Version Changer** kannst du aus einer Liste die gewünschte Minecraft-Version auswählen.
- Bestätige deine Auswahl, damit der Server mit der neuen Version installiert wird.

<Warning>
  Beachte, dass beim Ändern der Version bestehende Welten oder Plugins möglicherweise nicht mehr kompatibel sind. Sichere daher vorher deine Daten\!
</Warning>

<Info>
    Die Änderung der Version dauert in der Regel nur wenige Minuten. Danach kannst du dich direkt mit der neuen Minecraft-Version verbinden.
</Info>

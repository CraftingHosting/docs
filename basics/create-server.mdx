## Server erstellen

So erstellst du deinen eigenen Server Schritt für Schritt.

### Account anlegen

- Gehe ins Dashboard und gib deinen **Benutzernamen**, **E-Mail**, **Passwort** und (optional) den **Referral Code** eines Freundes ein.
- Klicke auf **Register**.

### Server erstellen

- Klicke im Dashboard auf **Server**.
- Klicke auf den blauen Button **Create Server**.
- Lege deinen **Servernamen** fest.
- Wähle bei **Software** die Option **Minecraft**.
- Scrolle nach unten und klicke erneut auf den blauen Button **Create Server**.

### Panel öffnen

- Klicke auf den blauen Button **🛠️**.
- Melde dich mit dem gleichen **Benutzernamen** und **Passwort** an, die du auch auf der Website verwendet hast.
- Klicke auf den blauen Button **Login**.

### Server verwalten

- Dein Server wird nun angezeigt.
- Klicke auf **Manage Server**, um ihn zu verwalten und zu starten.
